# CSS Masterclass

> A self-paced, beginner-friendly deep-dive into pure CSS — built for a teacher and her student.

Welcome! This course takes the student from "I have heard of CSS" to "I can confidently style a responsive website from scratch." Every lesson explains **what** to do, **why** it works, and uses a simple **analogy** the student can picture in her head.

After pure CSS is mastered, the course finishes with a roadmap into **Tailwind CSS**.

## What the student can build by the end

After finishing all 50 lessons and the 7 projects, the student will be able to:

- Style any HTML page using only CSS.
- Build pixel-perfect responsive layouts with **Flexbox** and **Grid**.
- Animate interfaces with `transform`, `transition` and `@keyframes`.
- Reorganize the same design into a **Tailwind** project using a translation guide.

## A teaching tip

Every lesson follows the same rhythm:

1. **Analogy** — a picture in her head.
2. **Why it matters** — the real-world reason it exists.
3. **How it works** — the smallest possible example.
4. **Try it yourself** — instructions for a small `.html` file she creates and edits.
5. **Common mistakes** — the pitfalls beginners hit.
6. **Debugging tips** — what to do when it looks wrong.
7. **Quiz** — a 3-question check.
8. **Mini challenge** — a slightly harder task.
9. **What's next** — link to the next lesson.

If she keeps that rhythm, she will never feel lost.

Happy teaching!
